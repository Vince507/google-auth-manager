## üöÄ Getting Started

1. Open your web browser and navigate to [myaccount.google.com/connections](https://myaccount.google.com/connections).
2. Press `Ctrl+A` to select all text on the page, then press `Ctrl+C` to copy it.
3. Open Google Auth Manager and paste the copied text into the provided text area.
4. Click the "Parse" button to extract the information.
5. Click the links generated to access each service's settings and backup methods.
